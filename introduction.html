Introduction to LavaLite

LavaLite is a lightweight PHP MVC framework designed for rapid development, simplicity, and modern coding practices. It is inspired by LavaLust, but focuses on **speed, minimal setup, and flexibility**.

What is LavaLite?

LavaLite provides:

  • Fluent database and query builder
  • Built-in security features like CSRF protection and session management
  • Simple routing with optional parameters
  • Easy integration of helper functions and modular components
  • Lightweight footprint for faster page load
LavaLite is perfect for small to medium PHP projects where you want speed without compromising structure.

System Requirements

  • PHP 8.1 or higher
  • Composer (optional, if using dependencies)
  • Web server (Apache, Nginx) or PHP built-in development server
  • VSCode (recommended) or any code editor
  • Git (optional, for clone method)
You can use the PHP development server without installing Apache or Nginx.

Installation Overview

LavaLite can be installed in two ways:

  • Using Git clone (recommended for development)
  • Manual download of the ZIP package

Install via Git Clone

Steps:


# Open VSCode terminal and navigate to your web folder
cd C:/xampp/htdocs    # Example for XAMPP on Windows

# Clone the repository
git clone https://github.com/ronmarasigan/LavaLite.git

# Navigate into project
cd LavaLite

After cloning, the project folder will contain the entire framework ready to run.

Manual Download

Steps:


# Download the ZIP package from GitHub
https://github.com/YourUsername/LavaLite/archive/refs/heads/main.zip

# Extract the ZIP into your web folder
C:/xampp/htdocs/LavaLite

# Navigate into the project folder
cd LavaLite

Running LavaLite Using VSCode Terminal

You can run LavaLite using the PHP built-in development server:


# Make sure you are in the project root (where index.php is located)
cd C:/xampp/htdocs/LavaLite

# Start PHP development server
php -S localhost:3000

Open your browser and navigate to:


http://localhost:3000

Optional: Using VSCode Live Server Extension

You can also use the Live Server extension for VSCode if you prefer an auto-refresh development experience, but PHP built-in server is recommended for PHP apps.
